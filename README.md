## [Japanese]
* 課題条件：Monaca + Firebase
* プロジェクト名：私の足跡

* 内容：
自分がどのような活動をしてきたのか、また現在どのような活動を進めているのかを記録するプログラムを作成することで、効率的な履歴書管理およびプロジェクト管理が可能になると考えた。

以下は、Cyber Security における CIA アプローチの手順として整理する。

| Type | 説明 |
| -- | -- |
| C (Confidentiality)|閲覧できる対象を限定する。<br>理由：個人情報への脅威が比較的少ない事例であると判断したため
I (Integrity) |	無整合性を確保するため、該当する履歴の作成は本人のみに限定して可能とする
A (Availability) | Firebase を使用して可用性を維持する

## [Korean]

* 과제 조건: Monaca + Firebase

* 프로젝트 명: 내 발자취

* 내용: 내가 어떤 활동을 했는지, 어떤 활동을 진행하고 있는지에 대한 프로그램을 작성하면 효율적인 이력서 관리, 프로젝트 관리가 가능하다고 생각함.

 다음은 Cyber Security에 대한 CIA접근 절차로 정리합니다.

| Type | 설명 |
| -- | --|
| C (Confidentiality) | 읽는 것은 모두에 한정한다.<br> 사유 : 개인정보의 위협이 적은 사례로 판단 |
| I (Integrity) | 무결성을 위해 해당 이력을 작성하는것을 본인에 한정하여 작성이 가능하게 설정한다 |
| A (Availability) | Firebase로 가용성을 유지한다|
